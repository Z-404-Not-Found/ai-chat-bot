## v1.0.0

发布日期：2026-04-09  
版本定位：首次完整版发布（毕设正式版）

### 核心功能

- 桌面端 AI 聊天：支持基于 Electron + Vue 的本地桌面聊天体验。
- 角色系统：支持角色新增、编辑、删除，支持自定义系统提示词（System Prompt）。
- 会话管理：支持按角色管理会话、会话重命名、会话删除与预览。
- 消息持久化：用户与助手消息本地落库（SQLite），支持历史记录回溯。
- 流式对话：支持流式输出、过程展示、请求中断与流式结束状态处理。
- Markdown 渲染：支持代码高亮、主题适配与内容安全净化。
- 对话长图导出：支持聊天记录一键导出为 PNG。
- AI 参数配置：支持 API Key、Base URL、Model、上下文条数、思考模式配置。
- 主题切换：支持多主题预设与明暗模式切换。
- 数据同步：支持账号注册/登录、手动同步、增量上传与拉取。

### 工程能力

- 安全 IPC 通信与 preload 白名单 API 设计。
- 主进程/渲染进程日志体系，便于排查问题。
- 跨平台构建支持（Windows/macOS/Linux）。
- 支持自动更新相关产物（安装包、`latest.yml`、`*.blockmap`）。

### 升级说明

- `v1.0.0` 为首次完整功能版本，建议全新安装体验。
- 如从早期版本覆盖升级，建议先备份本地配置与聊天数据。
